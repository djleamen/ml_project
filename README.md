# Machine Learning Project

This repository contains coursework from **AI and Machine Learning Algorithms and Techniques** by Microsoft on Coursera. The projects cover fundamental and advanced machine learning concepts across supervised learning, unsupervised learning, reinforcement learning, and deep learning.

## Course Structure

### Supervised Learning

- **p1_linear_regression.py** - Simple linear regression to predict house prices based on square footage
- **p2_logistic_regression.py** - Binary classification to predict student pass/fail based on study hours
- **p3_decision_trees.py** - Decision tree classifier for student performance prediction using study hours and exam scores
- **p4_comparing_models.py** - Comparison of logistic regression and decision tree classifiers
- **p5_metrics_crossvalid.py** - Model evaluation using metrics (accuracy, precision, recall, F1) and cross-validation techniques
- **p6_backward_elim.py** - Backward elimination for feature selection
- **p7_forward_selection.py** - Forward selection for feature selection
- **p8_lasso.py** - LASSO regression with L1 regularization for feature selection
- **p9_feature_selection.py** - Comprehensive comparison of backward elimination, forward selection, and LASSO

### Unsupervised Learning

- **p10_kmeans_clustering.py** - K-Means clustering for customer segmentation based on income and spending
- **p11_dbscan_clustering.py** - Density-based clustering (DBSCAN) for customer segmentation
- **p12_clustering_visualization.py** - Visualization of K-Means and DBSCAN clustering results
- **p13_dimensionality_reduction.py** - PCA and t-SNE for dimensionality reduction
- **p14_clustering_dimensionality_red.py** - Combining clustering with dimensionality reduction techniques
- **p15_unsupervised.py** - Comprehensive unsupervised learning analysis

### Reinforcement Learning

- **p16_qlearning_policy_gradient.py** - Q-Learning and policy gradient methods in a grid environment
- **p17_eval_metrics.py** - Evaluation metrics for reinforcement learning agents
- **p18_ml_paradigms.py** - Comparison of supervised, unsupervised, and reinforcement learning paradigms

### Deep Learning & Neural Networks

- **az_neural_networks.py** - Azure Machine Learning integration with neural networks
- **p19_neural_network.py** - Neural network for Fashion MNIST classification
- **p20_compare_nn_architecture.py** - Comparison of different neural network architectures
- **p21_pnns_cnns_rnns.py** - Implementation of Feedforward Neural Networks (FNNs), Convolutional Neural Networks (CNNs), and Recurrent Neural Networks (RNNs)
- **p22_gans_autoencoders.py** - Generative Adversarial Networks (GANs) and Autoencoders
- **p23_deep_learning_genai.py** - Deep learning for generative AI applications
- **p24_deep_learning_analysis.py** - Deep learning analysis on the Pima Indians Diabetes dataset

## Dataset

The `data/` directory contains the CIFAR-10 dataset used for image classification tasks in the deep learning modules.
